# Clean Architect Baseline Checklist

## Scope

- `/`
- `/builder`
- `/connectors`
- `/marketplace`
- `/mcp`
- `/runs`
- `/settings`

## Visual Acceptance

- Background uses neutral dark zinc look with high-contrast text.
- Borders stay subtle and consistent (1px style).
- Primary accent remains indigo/violet, without neon-heavy glow.
- Typography uses Geist Sans and Geist Mono consistently.
- Hover/focus states are precise and fast, without distracting effects.

## Functional Safety

- Builder interactions remain intact: add node, drag, connect, context menu, command palette.
- Keyboard shortcuts remain intact in builder.
- Theme defaults to dark and can be switched in Settings.
- No regressions in sidebar navigation and main dashboard flows.
